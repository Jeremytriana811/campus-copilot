# Campus Copilot

Campus Copilot is a grounded academic copilot for advising Q&A, degree planning, and schedule building.

## Design principles
- Official school documents are the source of truth
- Answers must be grounded and cited
- Unsupported questions should refuse
- Degree planning uses structured rules
- Schedule generation uses constraint solving
- Student sentiment is non-authoritative context only
- Architecture is local-first and cloud-ready
